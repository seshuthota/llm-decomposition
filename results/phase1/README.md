# Phase 1 Results

This directory stores the raw outputs for the first local experiment cycle on `Qwen/Qwen3-0.6B-Base`.

## Read These First

- [run_index.md](run_index.md): concise human-readable index of all runs
- [phase1_summary.md](phase1_summary.md): generated summary table from metrics
- [../../docs/experiments/phase1_results.md](../../docs/experiments/phase1_results.md): curated interpretation and decision

## Run Directories

Each run folder contains:

- `resolved_config.json`: exact run configuration after resolution
- `execution_status.json`: execution outcome
- `metrics.json`: top-level metrics and memory accounting
- `layer_errors.json`: layerwise error summaries
- `residual_profiles.json`: residual structure summaries
- `notes.md`: concise human-readable note for that run

## Preparation Summaries

- `phase1_preparation_summary.json`: generated by the original phase runner
- `phase1_summary.json` and `phase1_summary.md`: generated aggregate summaries across runs

Related preparation-only folders outside this directory:

- `results/phase1_baselines/`
- `results/phase1_repair/`

Those contain manifest-preparation summaries, not the canonical run outputs.
