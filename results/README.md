# Results

This directory stores raw experiment outputs, generated summaries, and execution logs.

## Layout

- [phase1/README.md](phase1/README.md): Phase 1 result layout and links
- [phase1/run_index.md](phase1/run_index.md): human-readable index of Phase 1 runs
- [phase1/phase1_summary.md](phase1/phase1_summary.md): generated summary table from metrics
- [logs/README.md](logs/README.md): execution log naming and usage

## Canonical Reading Order

For interpretation, use:

1. [phase1/run_index.md](phase1/run_index.md)
2. [phase1/phase1_summary.md](phase1/phase1_summary.md)
3. [../docs/experiments/phase1_results.md](../docs/experiments/phase1_results.md)
4. [../docs/experiments/experiment_journal.md](../docs/experiments/experiment_journal.md)

The `results/` tree is the raw source of truth for run outputs. The `docs/` tree is the curated narrative layer built on top of those outputs.
